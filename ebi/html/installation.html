Installation

Technology background

Dasty3 has two main modules, retrieval and visualization. The retrieval module runs on the user's web browser. It retrieves annotations from DAS servers and makes them available to the visualization module. The visualization module that provides the visual interface. The retrieval module is based on JSDAS and uses a PHP proxy that can easily be ported to other languages. The visualization module is build upon JQuery and based on AJAX.

Dasty3 makes use of standards-based presentation using HTML and Cascading Style Sheets (CSS), dynamic display and interaction using the Document Object Model (DOM), asynchronous data retrieval using the XMLHttpRequest function and JavaScript to bind everything together. Once the information has been retrieved by the visualization module, it is cached, eliminating the need for further DAS requests for most visual manipulations.

Requirements

  • Modern browser: Firefox 3.6+, Chrome 6.0+, Safari 5.0 or IE 8+
  • PHP5 with cURL support and Apache2

Set-up Instructions

  1. Download the lastest version of Dasty3. You can find it here.
  2. Unzip the downloaded file.
  3. Copy the unzipped folder into the htdocs folder in your Apache installation.
  4. Open your browser and go to http://<your server location>/Dasty/client. You should now see Dasty3 homepage.
  5. You are ready to go!
